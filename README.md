### Git Commit Convention
 
| 명칭       | 기능                                               |
|:--------:|--------------------------------------------------|
| Docs     | 문서 수정 (readme.md)                                |
| Set      | 프로젝트 세팅                                          |
| Feat     | 새로운 기능 추가                                        |
| Fix      | 버그 수정                                            |
| Design   | CSS 등 사용자 UI 수정                                  |
| Style    | 코드 formatting(enter같은), 세미콜론(;) 누락, 코드 변경이 없는 경우 |
| Refactor | 코드 리팩토링 (클린 코드로 개선)                              |
| Test     | 테스트 코드, 리팽토링 테스트 코드 추가                           |
|          |                                                  |
|          |                                                  |
